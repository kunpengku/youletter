git 初学

2018年8月7日

git 官网 https://git-scm.com/

SCM - Software Configuration Management.

查看 git 版本

git 的所有操作几乎都是本地的。

git 是完整的记录你的变化。几乎不可能有漏掉。通过对使用SHA-1 hash文件内容进行求和验证。

文件在git中有三种状态
committed, modified, and staged

配置git用户名和邮箱

git config –global user.name “fupeng”

git config –global user.email “fupeng@126.com“

查看配置

git config –list

使用git的两种场景，一是从本地已经存在的代码中创建仓库。二是从远程仓库拉取

从存在的代码中创建

git init
