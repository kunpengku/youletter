jenkins 指南旅行

2018年8月14日

下载然后运行Jenkins

  • 1， 下载 Jenkins
  • 2， 打开终端，进入下载目录
  • 3， 运行 java -jar jenkins.war –httpPort=8080
  • 4， 打开浏览器 http://localhost:8080
  • 5， 按照后面的步骤完成安装，需要在 ~/.jenkins/secrets/initialAdminPassword中找到激活密码，激活后，会安装一些插件。
  • 6， 装完插件后，会创建第一个管理员账户。
  • 7， 开始使用jenkins。

备注：
如果是用Mac OS的安装方式， 这个jenkins是杀不死的，杀死系统会又拉起来。启动和停止的命令如下

启动
sudo launchctl load /Library/LaunchDaemons/org.jenkins-ci.plist

停止
sudo launchctl unload /Library/LaunchDaemons/org.jenkins-ci.plist
