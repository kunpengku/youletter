修改Github commit的作者信息

2017年11月26日

在git项目的最上层目录，创建脚本 changename.sh
把其中的三个变量换成你需要的

  • OLD_EMAIL
  • CORRECT_NAME
  • CORRECT_EMAIL

执行 sh changename.sh

你本地的提交记录 作者信息就修改了。

然后git push -f 强制推送到远端即可。
