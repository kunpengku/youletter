spring-cloud是一个全家桶。而且还有各种不同的全家桶。

  • Angel

  • Brixton

  • Camden

  • Dalston

  • Edgware

  • Finchley

spring-cloud并不是一个jar包，而是一个jia包组合，或者说是一个配方。他们之间的不同有基于不同的 spring boot版本等。

Dalston 基于 Spring Boot 1.5.x.
Brixton 基于 Spring Boot 1.3.x.等
